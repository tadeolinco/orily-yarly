CODE DELIMITER START -> <STATEMENT> <CODE DELIMITER END>

STATEMENT  
    -> <DECLARATION DELIMITER>      <STATEMENT DELIMITER> <STATEMENT>
    -> <VARIABLE IDENTIFIER>        <STATEMENT DELIMITER> <STATEMENT>
    -> <INPUT>                      <STATEMENT DELIMITER> <STATEMENT>
    -> <OUTPUT>                     <STATEMENT DELIMITER> <STATEMENT>
    -> <CONDITIONAL DELIMITER>      <STATEMENT DELIMITER> <STATEMENT>
    -> <SWITCH DELIMITER>           <STATEMENT DELIMITER> <STATEMENT>
    -> <LOOP DELIMITER START>       <STATEMENT DELIMITER> <STATEMENT> // NOT YET
    -> <FUNCTION DELIMITER START>   <STATEMENT DELIMITER> <STATEMENT>
    -> <EXPRESSION>                 <STATEMENT DELIMITER> <STATEMENT>
    -> <CASTING EXPLICIT>           <STATEMENT DELIMITER> <STATEMENT> // NOT YET
    -> <RETURN_OPERATOR>            <STATEMENT DELIMITER> <STATEMENT>
    -> EMPTY


DELIMITER DELIMITER 
    -> <VARIABLE IDENTIFIER>
    -> <VARIABLE IDENTIFIER> <INITIALIZATION DELIMITER> <EXPRESSION>

INPUT 
    -> <VARIABLE IDENTIFIER>

OUTPUT 
    -> <EXPRESSION>

CONDITIONAL DELIMITER START
    -> <STATEMENT DELIMITER>
        <CONDITIONAL IF DELIMITER>
        <CONDITIONAL ELSE IF DELIMITER>
        <CONDTIONAL ELSE DELIMITER>
        <CONDTIONAL DELIMITER END>

CONDTIONAL IF DELIMITER
    -> <STATEMENT DELIMITER> 
            <STATEMENT>

CONDITIONAL ELSE IF DELIMITER
    -> <EXPRESSION> <STATEMENT DELIMITER>
            <STATEMENT>
        <CONDITIONAL ELSE IF DELIMITER>
    -> EMPTY

CONDTIONAL ELSE DELIMITER
    -> <STATEMENT DELIMITER> 
            <STATEMENT>

SWITCH DELIMITER
    -> <STATEMENT DELIMITER>
            <CASE DELIMITER>
            <DEFAULT CASE DELIMITER>
        <CONDTIONAL DELIMITER END>

CASE DELIMITER
    -> <LITERAL> <STATEMENT DELIMITER>
            <STATEMENT>
        <CASE DELIMITER>
    -> <LITERAL> <STATEMENT DELIMITER>
            <STATEMENT>
        <BREAK DELIMITER> <STATEMENT DELIMITER>
        <CASE DELIMITER>
    ->  EMPTY

DEFAULT CASE DELIMITER
    -> <STATEMENT DELIMITER>
        <STATEMENT>
    -> <STATEMENT DELIMITER>
        <STATEMENT>
        <BREAK DELIMITER>
    -> EMPTY

FUNCTION DELIMITER START
    -> <FUNCTION IDENTIFIER> <FUNCTION ARGUMENTS> <STATEMENT DELIMITER>
            <STATEMENT>
        <FUNCTION DELIMITER END>
        
RETURN OPERATOR
    -> <EXPRESSION>
